# Portfolio

A modern, responsive personal portfolio website built with Angular 19. Showcases professional experience, skills, projects, and contact information with smooth animations and an elegant dark theme.

## Installing / Getting started

To run the project locally:

```bash
npm install
npm start
```

The application will be available at `http://localhost:4200`.

## Developing

Clone the repository and install dependencies:

```bash
git clone https://github.com/Kvble/portfolio.git
cd portfolio
npm install
```

### Built with

- Angular 19
- TypeScript
- Angular CDK (Overlay for tooltips)
- Font Awesome Icons
- CountUp.js (animated counters)
- Typewriter Effect

## Features

### Profile Section
- Dynamic typewriter effect cycling through professional titles
- Rotating inspirational thoughts with fade transitions
- Profile image with elegant styling

### Skills Display
- Categorized skill badges (AI & LLMs, Backend, Cloud, Frontend, Databases)
- Icon-enhanced visual presentation
- Responsive grid layout

### Project Carousel
- Manual navigation with arrow controls
- Responsive sizing across all breakpoints (mobile, tablet, desktop)
- Hover effects with project overlays
- Direct links to project repositories

### Workflow Highlights
- Interactive workflow steps with tooltip descriptions
- Icon-based visual representation

### Additional Features
- Animated counters for years of experience and project count
- Client showcase carousel
- Contact section with social links
- Fully responsive design (mobile, tablet, desktop)
- Dark theme with glass-morphism effects
